# Project Name: Final Exam Project (MyWebApp)

## Project Description

MyWebApp is a responsive web application created to help users manage information in a simple and organized way. The focus of this project was learning how to structure a web application using the MVC (Model-View-Controller) pattern while keeping the interface clean and easy to use.

The application supports core data operations such as adding, updating, and displaying records, and it demonstrates how the frontend, backend, and database work together in a structured PHP environment.

---

## Project Tasks

**Task 1: Set up the development environment**

* Installed required tools (VS Code, XAMPP/VCASTLE, PHP, MySQL)
* Configured project folders and local server
* Connected project to GitHub

**Task 2: Design the application**

* Planned the MVC folder structure
* Designed basic page layout and data flow
* Identified required fields and database structure

**Task 3: Develop the frontend (View)**

* Built the interface using HTML and CSS
* Created forms for data entry and display
* Ensured the layout was readable and organized

**Task 4: Develop the backend (Controller)**

* Implemented controllers to handle user actions
* Routed form submissions to the correct logic
* Managed add, update, and delete operations

**Task 5: Build the data layer (Model)**

* Connected to the MySQL database
* Created functions for CRUD operations
* Retrieved and stored records securely

**Task 6: Connect to the database**

* Configured database connection
* Verified records could be inserted and retrieved
* Tested data flow between MVC components

**Task 7: Test the application**

* Ran the application through the local server
* Debugged routing and path issues
* Fixed form submission and display problems

**Task 8: Prepare for deployment**

* Verified the project runs locally without errors
* Cleaned up file structure
* Confirmed required files are included

**Task 9: Document the project**

* Created this README
* Added notes for setup and execution
* Documented skills learned

## Project Skills Learned

* PHP form handling and validation
* MySQL database connectivity

* Structuring a small web application from start to finish

## Development Process Used

**Agile-style workflow** — built the project in small steps, testing frequently and adjusting when issues came up. This project required flexibility because debugging and configuration took more time than expected in some areas.

## Notes
